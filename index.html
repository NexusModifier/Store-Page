About Us

Our Product

A React Admin dashboard app created using full stack with MERN acting as the back end stack with the use of Material UI on the front end. This dashboard is a modern admin board that focuses on providing detailed and clean organization for both businesses and personal use. Within the app, admin users will have access to files, data grids, beautiful graphs and UI as well as quick access to individual customers and team members for easy access to transactions and emails sent through a personalized chat system.

Our Goal

A task management dashboard app created using MERN and other tools such as JavaScript, Redux, and Material UI. CleanUp+ is a modern, simplified task management board that focuses on clean organization while providing single all-in-one widget access to tools used in day to day life. Such apps include Solo Task Board, Team Boards, Sketch App, File Management and embedded with rChat, a user chat interface built for organized communication that can also act as a standalone chat source.

Cultivation & Organics

Using React, Redux and Nodejs, rChat allows OxO users to be able to communicate between each other as well as other platforms including discord and twitter.
